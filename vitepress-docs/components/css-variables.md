# Variables

## GridLayout grid-lines
```sass
$grid-line-color: #000;
```


## GridItem `background-color`
```sass
$grid-item-bg-color: #720c0c;
```


## GridItem default `border-radius`.
```sass
$grid-item-border-radius: 8px;
```
